# John Deere Operations Center - Flags API

This archive contains markdown reference documents for all 14 endpoints in the Flags API documentation.

Source: https://developer.deere.com/dev-docs/flags

The Flags API is split into three resource groups: Flags, Flag Categories, and Flag Category Preferences.
